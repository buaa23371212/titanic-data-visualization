# 泰坦尼克号数据集 - 数据字典

## 数据集概述
泰坦尼克号数据集记录了1912年泰坦尼克号沉船事件中乘客的基本信息、船票信息和生存状态。该数据集包含891名乘客的详细信息，是数据分析和机器学习领域的经典数据集。

## 字段详细说明

### 1. PassengerId (乘客ID)
- **类型**: 整数 (Integer)
- **描述**: 乘客的唯一标识符
- **取值范围**: 1-891
- **示例**: 1, 2, 3

### 2. Survived (生存状态)
- **类型**: 布尔值 (Binary)
- **描述**: 乘客是否在沉船事件中幸存
- **取值**:
  - 0: 未幸存 (No)
  - 1: 幸存 (Yes)
- **示例**: 0, 1

### 3. Pclass (舱位等级)
- **类型**: 整数 (Integer)
- **描述**: 乘客的船票等级，反映社会经济地位
- **取值**:
  - 1: 一等舱 (Upper Class)
  - 2: 二等舱 (Middle Class)
  - 3: 三等舱 (Lower Class)
- **示例**: 1, 2, 3

### 4. Name (姓名)
- **类型**: 字符串 (String)
- **描述**: 乘客的完整姓名，包含称谓
- **格式**: "姓氏, 称谓. 名字"
- **示例**: "Braund, Mr. Owen Harris"

### 5. Sex (性别)
- **类型**: 字符串 (String)
- **描述**: 乘客的性别
- **取值**:
  - male: 男性
  - female: 女性
- **示例**: male, female

### 6. Age (年龄)
- **类型**: 浮点数 (Float)
- **描述**: 乘客的年龄（以年为单位）
- **取值范围**: 0.17-80
- **特殊值**: 存在缺失值（NaN）
- **示例**: 22.0, 38.0, 26.0

### 7. SibSp (兄弟姐妹/配偶数量)
- **类型**: 整数 (Integer)
- **描述**: 乘客在船上的兄弟姐妹或配偶的数量
- **取值范围**: 0-8
- **定义**:
  - 兄弟姐妹: 兄弟、姐妹、继兄弟姐妹
  - 配偶: 丈夫、妻子（情人不算）
- **示例**: 1, 0, 3

### 8. Parch (父母/子女数量)
- **类型**: 整数 (Integer)
- **描述**: 乘客在船上的父母或子女的数量
- **取值范围**: 0-6
- **定义**:
  - 父母: 母亲、父亲
  - 子女: 儿子、女儿、继子女
  - 某些儿童只与保姆同行，因此Parch=0
- **示例**: 0, 2, 1

### 9. Ticket (船票编号)
- **类型**: 字符串 (String)
- **描述**: 船票编号
- **特点**: 编号格式不统一，可能包含字母和数字
- **示例**: "A/5 21171", "PC 17599"

### 10. Fare (票价)
- **类型**: 浮点数 (Float)
- **描述**: 乘客支付的票价
- **取值范围**: 0-512.3292
- **单位**: 英镑
- **特点**: 票价可能包含多个乘客的总费用
- **示例**: 7.25, 71.2833

### 11. Cabin (船舱号)
- **类型**: 字符串 (String)
- **描述**: 乘客的船舱编号
- **特点**: 大量缺失值，仅204名乘客有记录
- **格式**: 通常为字母+数字（如C85）
- **示例**: C85, C123, E46

### 12. Embarked (登船港口)
- **类型**: 字符串 (String)
- **描述**: 乘客登船的港口
- **取值**:
  - C: Cherbourg (瑟堡)
  - Q: Queenstown (皇后镇)
  - S: Southampton (南安普顿)
- **示例**: S, C, Q

## 数据质量分析

### 缺失值统计
- **Age**: 177个缺失值 (19.9%)
- **Cabin**: 687个缺失值 (77.1%)
- **Embarked**: 2个缺失值 (0.2%)

### 数据分布特征
- **生存率**: 约38.4%的乘客幸存
- **性别比例**: 男性577人 (64.8%)，女性314人 (35.2%)
- **舱位分布**: 一等舱216人，二等舱184人，三等舱491人

## 数据预处理建议

### 1. 缺失值处理
- **Age**: 使用均值/中位数填充，或基于其他特征预测
- **Cabin**: 由于缺失值过多，考虑创建"是否有船舱记录"的二元特征
- **Embarked**: 使用众数填充

### 2. 特征工程
- **家庭规模**: 创建FamilySize = SibSp + Parch + 1
- **是否独自旅行**: IsAlone = (SibSp + Parch == 0)
- **姓名处理**: 提取称谓(Title)作为新特征
- **票价分组**: 将票价分为几个等级区间

### 3. 数据类型转换
- **分类变量**: Sex, Embarked 转换为数值编码
- **数值变量**: Age, Fare 进行标准化处理
