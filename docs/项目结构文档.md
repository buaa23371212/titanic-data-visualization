# 泰坦尼克号数据可视化项目 - 文件结构文档

## 项目概述
本项目旨在对泰坦尼克号乘客数据集进行全面的数据可视化分析，通过交互式图表展示乘客生存率、人口统计特征、社会经济因素等关键信息。

## 技术栈
- **前端**: React + TypeScript + Tailwind CSS
- **数据可视化**: D3.js + Chart.js
- **数据处理**: Python + Pandas + NumPy
- **构建工具**: Vite
- **包管理**: npm/yarn

## 核心功能模块

### 1. 数据预处理模块
- 数据清洗和验证
- 缺失值处理
- 数据类型转换
- 数据标准化

### 2. 可视化图表模块
- **生存分析**: 整体生存率、按舱位/性别/年龄分组
- **人口统计**: 年龄分布、性别比例、家庭结构
- **社会经济**: 票价分布、舱位等级分析
- **交互功能**: 筛选、钻取、联动

### 3. 用户界面模块
- 响应式布局设计
- 主题切换支持
- 无障碍访问优化
- 移动端适配

## 数据字段说明

| 字段名 | 类型 | 说明 |
|--------|------|------|
| PassengerId | 整数 | 乘客ID |
| Survived | 布尔值 | 是否幸存 (0=否, 1=是) |
| Pclass | 整数 | 舱位等级 (1=一等舱, 2=二等舱, 3=三等舱) |
| Name | 字符串 | 乘客姓名 |
| Sex | 字符串 | 性别 |
| Age | 浮点数 | 年龄 |
| SibSp | 整数 | 兄弟姐妹/配偶数量 |
| Parch | 整数 | 父母/子女数量 |
| Ticket | 字符串 | 船票编号 |
| Fare | 浮点数 | 票价 |
| Cabin | 字符串 | 船舱号 |
| Embarked | 字符串 | 登船港口 |

## 预期成果

1. **交互式仪表板**: 提供全面的数据探索体验
2. **多维度分析**: 支持按不同维度筛选和对比
3. **响应式设计**: 适配不同设备屏幕
4. **数据洞察**: 揭示泰坦尼克号事件中的关键模式
