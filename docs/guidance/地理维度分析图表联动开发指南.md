# 地理维度分析图表联动开发指南

## 概述

本文档描述了如何实现地理维度分析中两个图表（SurvivalBarChart 和 PortDistributionMap）的联动交互功能。当用户点击任一图表中的地名时，两个图表将同步高亮显示对应的数据点。

## 功能需求

1. **点击交互**：
   - 点击 SurvivalBarChart 或 PortDistributionMap 中的某个地名时触发事件
   - 再次点击同一地名时，事件结束（取消高亮）

2. **联动效果**：
   - SurvivalBarChart：对应柱形高亮显示
   - PortDistributionMap：定位到对应地点并显示港口信息（相当于点击对应地点）

## 技术实现方案

### 1. 状态管理

在 `GeographicAnalysis` 组件中创建共享状态来管理当前选中的地点：

```tsx
// 在 GeographicAnalysis 组件中添加
const [selectedLocation, setSelectedLocation] = useState<string | null>(null);

// 处理地点选择事件
const handleLocationSelect = (location: string | null) => {
  if (selectedLocation === location) {
    // 取消选择
    setSelectedLocation(null);
  } else {
    // 选择新地点
    setSelectedLocation(location);
  }
};
```

### 3. 实现高亮效果

#### SurvivalBarChart 高亮实现

在 `SurvivalBarChart.tsx` 中添加高亮逻辑：

```tsx
// 在 generateBars 函数中添加高亮逻辑
const generateBars = () => {
  return data.map((d, i) => {
    const isSelected = selectedLocation === d.category;
    const barOpacity = isSelected ? 0.9 : 0.7;
    const strokeWidth = isSelected ? 2 : 0;
    
    return (
      <g key={d.category} transform={`translate(${x}, 0)`}>
        {/* 未生存部分 */}
        <rect
          width={barWidth}
          height={notSurvivedHeight}
          y={notSurvivedY}
          fill={colors.notSurvived}
          opacity={barOpacity}
          stroke={isSelected ? colors.accent : 'none'}
          strokeWidth={strokeWidth}
          cursor="pointer"
          onClick={(e) => {
            onDataPointClick?.(d, e);
            onLocationSelect?.(d.category);
          }}
        />
        
        {/* 生存部分 */}
        <rect
          width={barWidth}
          height={survivedHeight}
          y={yScale(d.survived)}
          fill={colors.survived}
          opacity={barOpacity}
          stroke={isSelected ? colors.accent : 'none'}
          strokeWidth={strokeWidth}
          cursor="pointer"
          onClick={(e) => {
            onDataPointClick?.(d, e);
            onLocationSelect?.(d.category);
          }}
        />
        
        {/* 总数标签 */}
        <text
          x={barWidth / 2}
          y={yScale(d.total) - 8}
          textAnchor="middle"
          fontSize="10"
          fill={isSelected ? colors.accent : colors.text}
          fontWeight="bold"
          opacity={0.9}
        >
          {d.total}
        </text>
      </g>
    );
  });
};
```

const LocationFocus: React.FC<{ selectedLocation: string | null }> = ({ selectedLocation }) => {
  const map = useMap();

      if (port) {
        // 平滑移动到选中的港口

### 5. 更新 GeographicAnalysis 组件

```tsx
function GeographicAnalysis() {
  const { passengers, summaryStats } = usePassengerData();
  const [selectedLocation, setSelectedLocation] = useState<string | null>(null);
  
  const handleLocationSelect = (location: string | null) => {
    if (selectedLocation === location) {
      setSelectedLocation(null);
    } else {
      setSelectedLocation(location);
    }
  };
  
  return (
    <div className="space-y-6">
      <div className="text-center">
        <h2 className="text-2xl font-bold text-gray-900 dark:text-white mb-2">地理维度分析</h2>
        <p className="text-gray-600 dark:text-gray-400">登船港口分布、票价与港口关系分析</p>
      </div>
      <div className="grid grid-cols-1 lg:grid-cols-2 gap-6">
        <SurvivalBarChart
          category="byEmbarked"
          title="登船港口 vs 生存率"
          subtitle="不同港口乘客的生存情况对比"
          width={500}
          height={350}
          selectedLocation={selectedLocation}
          onLocationSelect={handleLocationSelect}
        />
        <CardContainer title="港口分布地图">
          <PortDistributionMap 
            passengers={passengers}
            portStats={summaryStats?.byEmbarked}
            selectedLocation={selectedLocation}
            onLocationSelect={handleLocationSelect}
          />
        </CardContainer>
      </div>
    </div>
  );
}
```

## 技术要点

1. **状态同步**：使用 React 状态管理确保两个图表的状态一致性
2. **性能优化**：使用 `useMemo` 和 `useCallback` 避免不必要的重渲染
3. **用户体验**：添加平滑动画和视觉反馈
4. **可访问性**：确保键盘导航和屏幕阅读器支持

## 扩展功能建议

1. **悬停效果**：鼠标悬停时显示预览效果
2. **多选模式**：支持同时选择多个地点进行比较
3. **数据过滤**：根据选择的地点过滤其他图表的数据
4. **动画效果**：添加更丰富的过渡动画

## 注意事项

1. 确保组件 props 的向后兼容性
2. 处理边界情况（如空数据、无效地点等）
3. 优化移动端触摸交互体验
