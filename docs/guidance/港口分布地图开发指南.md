# 港口分布地图开发指南

## 项目概述
本指南详细说明如何在泰坦尼克号数据可视化项目中实现港口分布地图功能。

## 技术选型建议

### 地图可视化方案
**推荐使用：Leaflet.js + React-Leaflet**

**理由：**
- **轻量级**：Leaflet是轻量级的开源地图库，适合简单的港口位置展示
- **React集成**：React-Leaflet提供React组件封装，与项目架构契合
- **免费开源**：无需API密钥，避免商业使用限制
- **易于定制**：可以自定义标记、弹窗等交互元素

**备选方案：**
- **Google Maps API**：功能强大但需要API密钥和可能产生费用
- **Mapbox GL JS**：现代化但相对复杂，适合更复杂的地图需求

### 第三步：创建港口统计工具函数
```typescript
export const calculatePortStats = (passengers: Passenger[], portCode: string): PortStats => {
  const portPassengers = passengers.filter(p => p.embarked === portCode);
  const survived = portPassengers.filter(p => p.survived).length;
  
  return {
    totalPassengers: portPassengers.length,
    survived,
    survivalRate: portPassengers.length > 0 ? survived / portPassengers.length : 0,
    averageFare: portPassengers.reduce((sum, p) => sum + p.fare, 0) / portPassengers.length,
    classDistribution: {
      1: portPassengers.filter(p => p.pclass === 1).length,
      2: portPassengers.filter(p => p.pclass === 2).length,
      3: portPassengers.filter(p => p.pclass === 3).length
    }
  };
};
```

### 第四步：实现地图组件
创建 `src/components/charts/PortDistributionMap.tsx`：

```typescript
import React from 'react';
import { MapContainer, TileLayer, Marker, Popup } from 'react-leaflet';
import { PORTS } from '../../utils/constants';
import { calculatePortStats } from '../../utils/portUtils';
import { Passenger } from '../../types/passenger';

interface PortDistributionMapProps {
  passengers: Passenger[];
  className?: string;
}

export const PortDistributionMap: React.FC<PortDistributionMapProps> = ({
  passengers,
  className = 'h-96'
}) => {
  return (
    <div className={className}>
      <MapContainer 
        center={[52.0, -3.0]} 
        zoom={5} 
        style={{ height: '100%', width: '100%' }}
      >
        <TileLayer
          url="https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png"
          attribution='&copy; <a href="https://www.openstreetmap.org/copyright">OpenStreetMap</a> contributors'
        />
        
        {PORTS.map(port => {
          const stats = calculatePortStats(passengers, port.code);
          
          return (
            <Marker key={port.code} position={port.coordinates}>
              <Popup>
                <div className="min-w-48">
                  <h3 className="font-bold text-lg">{port.fullName}</h3>
                  <div className="mt-2 space-y-1 text-sm">
                    <p>总乘客数: {stats.totalPassengers}</p>
                    <p>生存人数: {stats.survived}</p>
                    <p>生存率: {(stats.survivalRate * 100).toFixed(1)}%</p>
                    <p>平均票价: £{stats.averageFare.toFixed(2)}</p>
                  </div>
                </div>
              </Popup>
            </Marker>
          );
        })}
      </MapContainer>
    </div>
  );
};
```
