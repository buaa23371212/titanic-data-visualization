# 生存分析页面重做开发指南

## 概述

本文档描述了如何重做生存分析页面，将传统的柱状图分析改为点阵可视化，并实现多维度过滤器。新的设计将提供更直观的数据探索体验，让用户能够以点的形式查看每个乘客的生存状态，并通过多维度过滤器进行精细筛选。

## 功能需求

### 1. 点阵可视化

#### 数据接口定义
```typescript
// 乘客数据接口
interface Passenger {
  PassengerId: number;
  Survived: number; // 0: 死亡, 1: 生存
```

#### 可视化需求详情

- **均匀分布布局**：将每个乘客以点的形式均匀铺在页面中
- **筛选状态标识**：
  - 被筛选选中的乘客：显示彩色（生存：绿色，死亡：红色），可交互
  - 未被筛选选中的乘客：显示灰色且不可点击
- **生存状态标识**：用不同颜色区分生存（绿色）和死亡（红色）的乘客
- **交互功能**：
  - 鼠标悬停显示乘客详细信息（仅对筛选选中的乘客有效）
  - 点击选中单个乘客（仅对筛选选中的乘客有效）
  - 框选多个乘客进行批量操作（仅对筛选选中的乘客有效）

#### 布局实现优先级
1. **第一阶段**：实现均匀分布布局（grid）
2. **第二阶段**：可选实现数据相似度聚类布局（cluster）

### 2. 多维度过滤器
参考淘宝搜索栏的商品筛选器，实现以下维度的过滤：

#### 基础维度
- **生存状态**：生存/死亡/全部
- **性别**：男性/女性/全部
- **舱位等级**：一等舱/二等舱/三等舱/全部

#### 人口统计维度
- **年龄分组**：儿童(0-12岁)/青少年(13-17岁)/成人(18-59岁)/老人(60+岁)/全部
- **家庭规模**：独自旅行/小型家庭(2-3人)/大型家庭(4+人)/全部

#### 社会经济维度
- **票价分组**：低价票(0-10)/中等票价(10-30)/高价票(30+)/全部
- **登船港口**：瑟堡/皇后镇/南安普顿/全部

#### 高级筛选
- **年龄范围滑块**：可拖动选择年龄范围
- **票价范围滑块**：可拖动选择票价范围
- **搜索框**：支持按姓名模糊搜索

### 3. 过滤器交互特性
- **多选支持**：每个维度支持多选
- **即时过滤**：选择过滤器后实时更新点阵显示
- **筛选结果统计**：显示当前筛选条件下的乘客总数和生存率
- **清除筛选**：一键清除所有筛选条件
- **筛选历史**：支持撤销/重做筛选操作

### 2. 点阵布局算法

#### 均匀分布算法
```typescript
// 使用网格布局均匀分布点
const createGridLayout = (data: Passenger[], containerWidth: number, containerHeight: number) => {
  const pointsPerRow = Math.ceil(Math.sqrt(data.length));
  const spacingX = containerWidth / (pointsPerRow + 1);
  const spacingY = containerHeight / (pointsPerRow + 1);
  
  return data.map((passenger, index) => {
    const row = Math.floor(index / pointsPerRow);
    const col = index % pointsPerRow;
    
    return {
      passenger,
      x: (col + 0.5) * spacingX,
      y: (row + 0.5) * spacingY,
      size: 6 // 点的大小
    };
  });
};
```

**性能考虑与优化方案**

聚类算法在前端实时计算确实可能带来较大的性能开销，特别是对于K-means或层次聚类等复杂算法。以下是优化方案：

##### 方案一：Python预处理（推荐）

```python
# 在数据预处理阶段计算聚类结果
import pandas as pd
import numpy as np
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler

def precompute_clusters():
    # 加载泰坦尼克号数据集
    df = pd.read_csv('data/raw/Titanic-Dataset.csv')
    
    # 选择聚类特征
    features = ['Pclass', 'Age', 'Fare', 'SibSp', 'Parch']
    
    # 处理缺失值
    df_clean = df[features].fillna({
        'Age': df['Age'].median(),
        'Fare': df['Fare'].median()
    })
    
    # 标准化特征
    scaler = StandardScaler()
    scaled_features = scaler.fit_transform(df_clean)
    
    # 使用K-means聚类
    kmeans = KMeans(n_clusters=5, random_state=42)
    clusters = kmeans.fit_predict(scaled_features)
    
    # 保存聚类结果到JSON文件
    df['cluster'] = clusters
    cluster_data = df[['PassengerId', 'cluster']].to_dict('records')
    
    # 保存到前端可访问的文件
    with open('src/data/passenger_clusters.json', 'w') as f:
        json.dump(cluster_data, f, indent=2)
    
    print(f"聚类完成，共生成{len(np.unique(clusters))}个簇")
```

##### 方案二：前端简化聚类
```typescript
// 使用简化的基于网格的聚类，避免复杂计算
const createSimpleClusterLayout = (data: Passenger[], containerWidth: number, containerHeight: number) => {
  // 基于特征值的简单分组
  const groups = data.reduce((acc, passenger) => {
    const groupKey = `${passenger.Pclass}-${Math.floor(passenger.Age_original || 0 / 10)}-${Math.floor(passenger.Fare || 0 / 20)}`;
    if (!acc[groupKey]) acc[groupKey] = [];
    acc[groupKey].push(passenger);
    return acc;
  }, {} as Record<string, Passenger[]>);
  
  // 在容器内均匀分布各组
  const groupKeys = Object.keys(groups);
  const groupsPerRow = Math.ceil(Math.sqrt(groupKeys.length));
  
  return groupKeys.flatMap((key, groupIndex) => {
    const group = groups[key];
    const row = Math.floor(groupIndex / groupsPerRow);
    const col = groupIndex % groupsPerRow;
    
    const groupX = (col + 0.5) * (containerWidth / groupsPerRow);
    const groupY = (row + 0.5) * (containerHeight / groupsPerRow);
    const groupRadius = Math.min(containerWidth, containerHeight) / (groupsPerRow * 3);
    
    // 在组内均匀分布点
    return group.map((passenger, pointIndex) => {
      const angle = (pointIndex / group.length) * 2 * Math.PI;
      const radius = groupRadius * (pointIndex === 0 ? 0 : 0.8); // 第一个点在中心
      
      return {
        passenger,
        x: groupX + Math.cos(angle) * radius,
        y: groupY + Math.sin(angle) * radius,
        size: 6,
        group: key
      };
    });
  });
};
```

```typescript
const useClusterWorker = (data: Passenger[]) => {
  const [clusters, setClusters] = useState<ClusterPoint[]>([]);
  
  useEffect(() => {

    };
    
    return () => worker.terminate();
  }, [data]);
  
  return clusters;
```

### 3. 过滤器状态管理

```typescript
interface FilterState {
  // 基础维度
  survivalStatus: ('survived' | 'died')[];
  gender: ('male' | 'female')[];
  passengerClass: (1 | 2 | 3)[];
  
  // 人口统计维度
  ageGroup: ('child' | 'teen' | 'adult' | 'senior')[];
  familySize: ('solo' | 'small' | 'large')[];
  
  // 社会经济维度
  fareGroup: ('low' | 'medium' | 'high')[];
  embarked: ('C' | 'Q' | 'S')[];
  
  // 高级筛选
  ageRange: [number, number];
  fareRange: [number, number];
  nameSearch: string;
}

const useFilterState = () => {
  const [filters, setFilters] = useState<FilterState>({
    survivalStatus: [],
    gender: [],
    passengerClass: [],
    ageGroup: [],
    familySize: [],
    fareGroup: [],
    embarked: [],
    ageRange: [0, 80],
    fareRange: [0, 512],
    nameSearch: ''
  });
  
  // 更新单个过滤器
  const updateFilter = (filterKey: keyof FilterState, value: any) => {
    setFilters(prev => ({ ...prev, [filterKey]: value }));
  };
  
  // 清除所有过滤器
  const clearAllFilters = () => {
    setFilters({
      survivalStatus: [],
      gender: [],
      passengerClass: [],
      ageGroup: [],
      familySize: [],
      fareGroup: [],
      embarked: [],
      ageRange: [0, 80],
      fareRange: [0, 512],
      nameSearch: ''
    });
  };
  
  return { filters, updateFilter, clearAllFilters };
};
```

### 4. 数据过滤逻辑

```typescript
const usePassengerFilter = (passengers: Passenger[], filters: FilterState) => {
  return useMemo(() => {
    return passengers.filter(passenger => {
      // 生存状态过滤
      if (filters.survivalStatus.length > 0) {
        const isSurvived = passenger.Survived === 1;
        if (!filters.survivalStatus.includes(isSurvived ? 'survived' : 'died')) {
          return false;
        }
      }
      
      // 性别过滤
      if (filters.gender.length > 0) {
        if (!filters.gender.includes(passenger.Sex.toLowerCase() as 'male' | 'female')) {
          return false;
        }
      }
      
      // 舱位等级过滤
      if (filters.passengerClass.length > 0) {
        if (!filters.passengerClass.includes(passenger.Pclass as 1 | 2 | 3)) {
          return false;
        }
      }
      
      // 年龄分组过滤
      if (filters.ageGroup.length > 0) {
        const age = passenger.Age_original || 0;
        let ageGroup: string;
        if (age <= 12) ageGroup = 'child';
        else if (age <= 17) ageGroup = 'teen';
        else if (age <= 59) ageGroup = 'adult';
        else ageGroup = 'senior';
        
        if (!filters.ageGroup.includes(ageGroup as any)) {
          return false;
        }
      }
      
      // 年龄范围过滤
      const age = passenger.Age_original || 0;
      if (age < filters.ageRange[0] || age > filters.ageRange[1]) {
        return false;
      }
      
      // 票价范围过滤
      const fare = passenger.Fare || 0;
      if (fare < filters.fareRange[0] || fare > filters.fareRange[1]) {
        return false;
      }
      
      // 姓名搜索过滤
      if (filters.nameSearch) {
        const searchTerm = filters.nameSearch.toLowerCase();
        const name = passenger.Name?.toLowerCase() || '';
        if (!name.includes(searchTerm)) {
          return false;
        }
      }
      
      return true;
    });
  }, [passengers, filters]);
};
```

### 5. 组件实现细节

#### PassengerScatterPlot 组件实现
```tsx
const PassengerScatterPlot: React.FC<PassengerScatterPlotProps> = ({
  passengers,
  filteredPassengerIds = [],
  width,
  height,
  layoutType = 'grid',
  onPassengerHover,
  onPassengerClick,
  onPassengerSelect
}) => {
  const layout = useScatterLayout(passengers, width, height, layoutType);
  const [hoveredPassenger, setHoveredPassenger] = useState<Passenger | null>(null);
  const [selectedPassengers, setSelectedPassengers] = useState<Set<number>>(new Set());
  
  // 判断乘客是否被筛选选中
  const isFiltered = (passengerId: number) => {
    return filteredPassengerIds.length === 0 || filteredPassengerIds.includes(passengerId);
  };
  
  // 获取点的颜色和样式
  const getPointStyle = (passenger: Passenger) => {
    const filtered = isFiltered(passenger.PassengerId);
    
    if (!filtered) {
      // 未被筛选选中的乘客：灰色，不可点击
      return {
        fill: '#9ca3af', // 灰色
        cursor: 'default',
        opacity: 0.5
      };
    }
    
    // 被筛选选中的乘客：根据生存状态显示颜色
    return {
      fill: passenger.Survived ? '#10b981' : '#ef4444',
      cursor: 'pointer',
      opacity: 1
    };
  };
  
  const handlePointClick = (passenger: Passenger) => {
    if (!isFiltered(passenger.PassengerId)) return; // 未被筛选选中的乘客不可点击
    
    const newSelected = new Set(selectedPassengers);
    if (newSelected.has(passenger.PassengerId)) {
      newSelected.delete(passenger.PassengerId);
    } else {
      newSelected.add(passenger.PassengerId);
    }
    setSelectedPassengers(newSelected);
    onPassengerClick?.(passenger);
    onPassengerSelect?.(Array.from(newSelected));
  };
  
  const handlePointHover = (passenger: Passenger | null) => {
    if (passenger && !isFiltered(passenger.PassengerId)) return; // 未被筛选选中的乘客不显示悬停
    
    setHoveredPassenger(passenger);
    onPassengerHover?.(passenger);
  };
  
  return (
    <div className="relative">
      <svg width={width} height={height}>
        {layout.map((point, index) => {
          const style = getPointStyle(point.passenger);
          const isSelected = selectedPassengers.has(point.passenger.PassengerId);
          
          return (
            <circle
              key={index}
              cx={point.x}
              cy={point.y}
              r={isSelected ? point.size * 1.5 : point.size} // 选中时放大
              fill={style.fill}
              opacity={style.opacity}
              cursor={style.cursor}
              stroke={isSelected ? '#3b82f6' : 'none'} // 选中时显示蓝色边框
              strokeWidth={isSelected ? 2 : 0}
              className="transition-all duration-200"
              onMouseEnter={() => handlePointHover(point.passenger)}
              onMouseLeave={() => handlePointHover(null)}
              onClick={() => handlePointClick(point.passenger)}
            />
          );
        })}
      </svg>
      
      {/* 乘客信息提示框 */}
      {hoveredPassenger && (
        <PassengerTooltip passenger={hoveredPassenger} />
      )}
    </div>
  );
};
```

#### 均匀分布布局实现（第一阶段）
```typescript
const useScatterLayout = (
  passengers: Passenger[], 
  containerWidth: number, 
  containerHeight: number, 
  layoutType: 'grid' | 'cluster' = 'grid'
): ScatterPoint[] => {
  return useMemo(() => {
    if (layoutType === 'grid') {
      return createGridLayout(passengers, containerWidth, containerHeight);
    }
    // cluster布局将在第二阶段实现
    return createGridLayout(passengers, containerWidth, containerHeight);
  }, [passengers, containerWidth, containerHeight, layoutType]);
};

const createGridLayout = (data: Passenger[], containerWidth: number, containerHeight: number): ScatterPoint[] => {
  if (data.length === 0) return [];
  
  const pointsPerRow = Math.ceil(Math.sqrt(data.length));
  const spacingX = containerWidth / (pointsPerRow + 1);
  const spacingY = containerHeight / (pointsPerRow + 1);
  
  return data.map((passenger, index) => {
    const row = Math.floor(index / pointsPerRow);
    const col = index % pointsPerRow;
    
    return {
      passenger,
      x: (col + 0.5) * spacingX,
      y: (row + 0.5) * spacingY,
      size: 6,
      isFiltered: true // 初始状态都为筛选选中
    };
  });
};
```
