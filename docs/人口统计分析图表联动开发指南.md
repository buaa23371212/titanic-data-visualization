# 人口统计分析图表联动开发指南

## 概述

本文档描述了如何实现人口统计分析中两个图表（FamilyStructureChart 和 AgeDistributionChart）的联动交互功能。当用户在家庭结构饼图中选择特定家庭规模时，年龄分布直方图将自动筛选显示对应家庭规模的年龄分布数据。

## 功能需求

1. **点击交互**：
   - 点击 FamilyStructureChart 饼图中的扇形选择对应家庭规模
   - 再次点击同一扇形取消选中状态
   - 没有选择时，AgeDistributionChart 显示所有数据

2. **数据过滤**：
   - 根据选中的家庭规模筛选 AgeDistributionChart 使用的数据
   - 筛选逻辑基于乘客的 FamilySizeGroup 字段

## 技术实现方案

```tsx
// 在 generatePie 函数中添加高亮逻辑
const generatePie = () => {
  const pieData = pie(data);
  
  return pieData.map((slice, index) => {
    const isSelected = selectedFamilySize === slice.data.label;
    const arcPath = arc(slice);
    const labelPosition = labelArc.centroid(slice);
    const percentage = ((slice.endAngle - slice.startAngle) / (2 * Math.PI) * 100).toFixed(1);
    
    // 高亮样式
    const fillOpacity = isSelected ? 1 : 0.8;
    const strokeWidth = isSelected ? 3 : 2;
    const strokeColor = isSelected ? colors.accent : colors.background;
    
    return (
      <g key={slice.data.label} transform={`translate(${chartWidth / 2}, ${chartHeight / 2})`}>
        {/* 饼图扇形 */}
        <path
          d={arcPath || ''}
          fill={pieColors[index % pieColors.length]}
          opacity={fillOpacity}
          stroke={strokeColor}
          strokeWidth={strokeWidth}
          cursor="pointer"
          onMouseEnter={(e) => onDataPointHover?.(slice.data, e)}
          onMouseLeave={(e) => onDataPointHover?.(null, e)}
          onClick={(e) => {
            onDataPointClick?.(slice.data, e);
            onFamilySizeSelect?.(slice.data.label);
          }}
        />
        
        {/* 百分比标签 */}
        {slice.data.value > 0 && (
          <text
            x={labelPosition[0]}
            y={labelPosition[1]}
            textAnchor="middle"
            dominantBaseline="middle"
            fontSize="12"
            fill={isSelected ? colors.accent : colors.text}
            fontWeight="600"
          >
            {percentage}%
          </text>
        )}
      </g>
    );
  });
};
```

### 4. 实现数据过滤逻辑

#### AgeDistributionChart 数据过滤

在 `AgeDistributionChart.tsx` 中添加数据过滤逻辑：

```tsx
// 修改 useMemo 中的数据生成逻辑
const chartData = useMemo(() => {
  if (!passengers || passengers.length === 0) return [];
  
  // 根据选中的家庭规模过滤数据
  let filteredPassengers = passengers;
  if (selectedFamilySize) {
    filteredPassengers = passengers.filter(passenger => {
      switch (selectedFamilySize) {
        case '独自旅行':
          return passenger.FamilySizeGroup === 'Solo';
        case '小型家庭':
          return passenger.FamilySizeGroup === 'Small';
        case '大型家庭':
          return passenger.FamilySizeGroup === 'Large';
        default:
          return true;
      }
    });
  }
  
  // 过滤掉年龄缺失的数据
  const validAges = filteredPassengers
    .filter(p => p.Age_original !== null && p.Age_original !== undefined)
    .map(p => p.Age_original as number);
  
  if (validAges.length === 0) return [];
  
  // 创建直方图分组
  const ageBins = d3.bin()
    .domain([0, 80]) // 年龄范围：0-80岁
    .thresholds(8) // 8个分组
    (validAges);
  
  return ageBins.map((bin, index, array) => {
    // 计算每个柱子的横轴标签：显示区间的起始值
    const axisLabel = index === 0 ? '0' : `${bin.x0}`;
    
    return {
      label: `${bin.x0}-${bin.x1}岁`,
      value: bin.length,
      percentage: bin.length / validAges.length,
      binStart: bin.x0 || 0,
      binEnd: bin.x1 || 0,
      axisLabel: axisLabel
    };
  });
}, [passengers, selectedFamilySize]); // 添加 selectedFamilySize 依赖
```

### 5. 更新 DemographicsAnalysis 组件

```tsx
function DemographicsAnalysis() {
  const [selectedFamilySize, setSelectedFamilySize] = useState<string | null>(null);
  
  const handleFamilySizeSelect = (familySize: string | null) => {
    if (selectedFamilySize === familySize) {
      setSelectedFamilySize(null);
    } else {
      setSelectedFamilySize(familySize);
    }
  };
  
  return (
    <div className="space-y-6">
      <div className="text-center">
        <h2 className="text-2xl font-bold text-gray-900 dark:text-white mb-2">人口统计分析</h2>
        <p className="text-gray-600 dark:text-gray-400">年龄分布、家庭结构、票价分布等维度分析</p>
        
        {/* 显示当前筛选状态 */}
        {selectedFamilySize && (
          <div className="mt-4 p-3 bg-blue-50 dark:bg-blue-900 rounded-lg">
            <span className="text-blue-700 dark:text-blue-300 font-medium">
              当前筛选: {selectedFamilySize} 的年龄分布
            </span>
          </div>
        )}
      </div>
      
      <div className="grid grid-cols-1 lg:grid-cols-2 gap-6">
        <CardContainer>
          <AgeDistributionChart
            title="年龄分布直方图"
            subtitle={selectedFamilySize ? `${selectedFamilySize}的年龄分布情况` : "乘客年龄分布情况"}
            width={500}
            height={350}
            selectedFamilySize={selectedFamilySize}
          />
        </CardContainer>
        <CardContainer>
          <FamilyStructureChart
            title="家庭结构分析"
            subtitle="点击扇形筛选对应家庭规模的年龄分布"
            width={500}
            height={350}
            selectedFamilySize={selectedFamilySize}
            onFamilySizeSelect={handleFamilySizeSelect}
          />
        </CardContainer>
      </div>
    </div>
  );
}
```

## 技术要点

1. **状态同步**：使用 React 状态管理确保两个图表的状态一致性
2. **数据过滤**：基于 FamilySizeGroup 字段进行精确筛选
3. **性能优化**：使用 `useMemo` 避免不必要的重计算
4. **用户体验**：添加视觉反馈和状态提示

## 数据映射关系

- "独自旅行" ↔ FamilySizeGroup = 'Solo'
- "小型家庭" ↔ FamilySizeGroup = 'Small' 
- "大型家庭" ↔ FamilySizeGroup = 'Large'
